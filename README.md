Project Title: Student Project Management System (SPMS)

Description:
This project helps colleges manage student project submissions, approvals, feedback, and progress tracking.
It includes secure login for Admin, Guide, and Student with role-based dashboards.

Features:
- Secure Login & Registration
- Topic Submission & Approval
- Duplicate Topic Detection
- Task Assignment & Progress Tracking
- Feedback & Notifications
- Dashboard with Charts
- Upload file 
- Deadline Reminders

Software / Technologies Used:
- Backend Framework : Flask (Python)
- Frontend          : HTML5, CSS3, JavaScript
- Template Engine   : Jinja2
- Database          : SQLite3
- Charts / Graphs   : Chart.js
- Email Service     : SMTP (Gmail)
- IDE               : Visual Studio Code
- Browser Tested    : Microsoft Edge

Hardware Requirements:
- Processor: Intel i3 or higher
- RAM: 4GB minimum
- Storage: 1GB free space
- OS: Windows 10 / 11

Software Requirements:
- Python 3.x
- Flask Framework
- SQLite3 Database
- Any Web Browser
